# Custom Fonts

This directory should contain your custom font files:

## SOONER Font
Place SOONER font files here:
- sooner.woff2 (recommended)
- sooner.woff (fallback)

## CAMOOD Font  
Place CAMOOD font files here:
- camood.woff2 (recommended)
- camood.woff (fallback)

If you don't have these custom fonts, you can:
1. Download them from the font provider
2. Use Google Fonts alternatives like:
   - Poppins for headings
   - Open Sans for body text
   - Montserrat for display text

To update the font fallbacks, edit `app/globals.css` and `tailwind.config.js`.
